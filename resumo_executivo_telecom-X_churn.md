#  Projeto de Análise de Churn (ETL/EDA)– TelecomX

##  Resumo Executivo

##### Este projeto teve como objetivo preparar os dados para identificar os principais fatores associados à evasão de clientes (churn) em uma empresa de telecomunicações, utilizando técnicas de **ETL (Extração, Transformação e Carga)** e **Análise Exploratória de Dados (EDA)**.

#####  Os dados foram extraídos a partir de uma API em formato JSON aninhado, posteriormente normalizados e tratados para garantir qualidade analítica. Durante a etapa de transformação, foram realizados ajustes de tipos de dados, tratamento de valores ausentes e padronização de variáveis categóricas e numéricas.

#####  A análise revelou que clientes com **contratos mensais**, **maior valor de cobrança mensal**, **menor tempo de permanência**, **ausência de serviços de suporte técnico** e **perfil etário mais elevado** apresentam maior propensão à evasão. Esses padrões indicam que fatores contratuais e percepção de valor exercem impacto significativo no comportamento de churn.

#####  Com base nos resultados, foram propostas recomendações estratégicas focadas em retenção, como incentivo a contratos de longo prazo, revisão de planos de maior custo, fortalecimento do suporte técnico e segmentação de ações por perfil de cliente.

#####  O projeto demonstra domínio prático de manipulação de dados com Python, organização de pipelines de dados e interpretação analítica orientada a negócio com potencial de evolução para aplicações preditivas.
